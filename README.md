# 팀 빌딩 플랫폼 서비스

React, Node JS, MongoDB를 활용한 IT 프로젝트 팀 빌딩 플랫폼 서비스 입니다.

## FEATURES

IT 프로젝트를 위해 필요한 팀원 모집글을 게시한다.
팀이 직접 구인할 수 있도록 개인의 소개글도 게시한다.
찜하기 기능을 통해 관심있는 아이템을 관리한다.
